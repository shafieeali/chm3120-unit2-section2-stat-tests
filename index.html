How to Use This Interactive App Accessibly

Navigation

The left navigation opens each lecture section, the practice lab, the full slide studio, and the class-note browser. The skip link moves keyboard users directly to the main workspace.

Slides and Notes

Slide and note images are paired with extracted text transcripts. The browser controls include previous, next, and print actions.

Interactive Work

Calculators and decision coaches provide live feedback. Practice answers are hidden until students press the reveal button for each individual card.

Text-first Mode

Text-first mode hides visual-heavy panels and keeps the transcripts, calculations, feedback, and practice cards available.
